Análisis de Datos con Pandas Pandas Matplotlib
Procesamiento de Datos

El proyecto utiliza Pandas para la lectura y procesamiento de archivos CSV. Los datos se cargan desde data/reportes/excel/empleados.csv y se procesan para generar estadísticas y reportes.

  • Lectura de CSV: pd.read_csv() con separador punto y coma (;)
  • Agrupación: groupby('Departamento') para calcular promedios por categoría
  • Estadísticas: Cálculo de sueldos promedio, edad promedio, valores máximos y mínimos
Generación de Gráficos

Con Matplotlib se generan gráficos estadísticos a partir de los datos procesados:

  • Gráfico de líneas: Visualización de sueldos promedio por departamento
  • Exportación: Los gráficos se guardan en data/reportes/graficos/ con timestamp
  • Incrustación: Los gráficos se convierten a Base64 para mostrarse en HTML
Visualización de Datos

Los DataFrames se convierten a tablas HTML estilizadas con Bootstrap:

  • Tablas responsivas: df.to_html() con clases Bootstrap
  • Datos en JSON: Exportación a formato JSON para APIs externas
  • Estadísticas dinámicas: Cálculo automático de métricas en tiempo real
Tecnologías utilizadas: Django 6.0 Pandas Matplotlib Bootstrap 5 drf-spectacular
